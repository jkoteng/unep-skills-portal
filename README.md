# UNEP Staff Skills Portal

## Project Description
An online portal to store and manage staff technical skills, developed as part of the UNEP system assessment.

## Setup Instructions
1. **Database:** Import the included `.sql` file into your local MySQL/phpMyAdmin environment.
2. **Files:** Place the project folder in your local server directory (e.g., `htdocs` for XAMPP).
3. **Run:** Access the portal at `http://localhost/unep_skills/index.php`.

## Credentials
- No login is required for this version; the staff listing and forms are accessible for review.
